# CHIP-8 Emulator

C implementation of CHIP-8

## Install

`mkdir build`

`cd build`

`cmake ..`

`make`

<br>

## Usage

`chip8 ..games/<game.ch8>`

## Speed

You can modulate emulation speed by giving a second param.

`chip8 ..games/<game.ch8> 120`
